What it is

DiffPriv is a collection of different defferential privacy algorithms. From the Laplace Mechanism to the simple Random Response mechanism, use differential privacy in your data easily with DiffPriv.

With DiffPriv, you can easily privatize your data in an instant. With quick and easy ways to differentially privatize your data, it makes differential privacy easy as pie (not 3.14159265358979323846264338327950288419716939937510582097494459230781640628620899862803

How to Use

There are several different ways you can implement diferential privacy in.

  • Random Response
  • Exponenential Mechanism
  • Laplace Mechanism

Those are a few of the mechanisms for differential privacy. Sadly, in DiffPriv, we have 2 of those. The Laplace Mechanism and Random Response (watch out for v0.0.3, where we will add more mechanisms).



Downloading DiffPriv

To get DiffPriv, open up your command prompt. Type:

python3 -m pip install DiffPriv==0.0.2

and you're done. You're ready to start using DiffPriv!

Random Response

The random response mechanism is one of the simplest ways to implement differential privacy. It will flip a coin (well, not literally. This is just an easier way to explain it) and if it lands on heads, it returns the truth. If it's tails, it will flip it again and this time, if it lands on heads, it returns true. If it is tails, it returns false. Here's a helpful diagram.



To use this mechaism, you use a function called random.

random(data)

Data is your dataset.



Laplace Mechansim

The Laplace Mechanism is a much more complex method, so I won't take the time to explain it. You can see a good explanation on the laplace mechanism both on Wikipedia and in this great book.But, if you want to use it, you can use it by calling the laplace function.

laplace(data*, file_name*, epsilon*, f*, sample_size, delta_f)

The first four are required. For a really great explanation of the parameters, view @q9i's Differential Privacy wiki page on the Laplace Mechanism on Github.
